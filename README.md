# 卒業課題
## 開発言語バージョン
 - rails 6.1.6
 - ruby 3.0.1

## 就業Termで学んだ技術
 - devise
 - AWS EC2

## カリキュラム外技術
 - simple_calender
 - ransack

 ## アプリケーションの実行手順
  ```
  $ git clone https://github.com/Jinteng-Yougui/graduation_app.git 
  $ cd graduation_app
  $ bundle install
  $ yarn install
  $ rails db:create && rails db:migrate
  $ rails s
  ```
